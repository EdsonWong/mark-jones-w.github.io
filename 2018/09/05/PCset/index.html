前言




虽然现在路由器广播无线信号的技术已经很成熟了，但是依旧无法保证数据连通的可靠性，即便是有线传输也一样。其次对于我们学生来说，有时候不一定真的需要路由器，电信的校园流量是不限量的，而且自2018年开始限速从60G改到了100G（目前9月份无论新老生办理新套餐都是这样）打一波广告：

故无论出自何种考虑，直接用网线连接电脑也是一种选择，在这里，也简单的写一点东西，可能不需要这个教程你也会，有兴趣就继续看下去吧。



第一步

首先连线。先关闭电脑的WiFi，然后将墙上的端口与电脑的端口用线相连接。会发现工具栏处出现了一个带小感叹号的类似电脑的一个小图标，点一下，显示的是无Internet，如果是正在识别中则等一小会儿。

然后就进行下一步了

网络和Internet设置 也可以在 开始 -> 控制面板 -> 网络和Internet 中找到。设置新的网络连接，则 开始 -> 控制面板 -> 网络和Internet ->设置新的连接或网络

第二步

如图说的很清楚，讲几个点。首先要把多余的连接删除，避免多次拨号导致未知错误，其次有些同学打不开这个设置，或者一直出错，建议去更新一下有线网卡驱动

更新网卡驱动如下。注意那个Realtek Wireless LAN是无线驱动网卡，一般同规格的另外一个就是有线网卡（图中就是上面那个）然后右击更新驱动程序。
